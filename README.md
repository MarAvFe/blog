## Installation

Run local server:

```bash
$ git clone git@github.com:maravfe/blog.git
$ cd blog
$ bundle install
$ bundle add webrick
$ bundle exec jekyll serve
```

Navigate to `localhost:4000`. You're Welcome, Fork and be Stargazer.
